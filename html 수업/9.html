    • HTML
    HTML은 하이퍼 텍스트 마크업 랭기지의 약자로 매우 쉽고 빠르게 웹 문서를 만들 수 있는 마크업 언어입니다.











    CSS
    css는 캐스캐딩 스타일 시트의 약자로 html 문서의 스타일을 정의하는 규약입니다.











    자바스크립트
    자바스크립트는 html 문서에서 사용할 수 있는 프로그래밍 언어로 html 문서에 다양한 효과를 줄 때 사용합니다.
